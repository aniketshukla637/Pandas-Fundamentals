# Pandas Fundamentals

## 📌 Overview
This repository covers the core fundamentals of the **Pandas** library in Python, used for data manipulation and analysis in Machine Learning workflows.

## 🛠️ Tech Stack
- Python
- Pandas
- Jupyter Notebook / Google Colab

## 🔑 Topics Covered
- Series and DataFrame basics
- Reading data (CSV, Excel, JSON)
- Indexing and selection (`loc`, `iloc`)
- Filtering and conditional selection
- Handling missing values
- Sorting and grouping data
- Basic data manipulation and cleaning

## 🎯 Learning Outcome
Built a strong foundation in using Pandas for data loading, cleaning, and manipulation — a core skill for any Data Science/ML pipeline.
